# geometric_libr (Библиотека геометрических вычислений)

## Общее описание решения

Данная библиотека предоставляет функции для вычисления площадей и периметров различных геометрических фигур. Библиотека написана на Python.

## Функции

### rectangle.py

#### area(a, b)

Вычисляет площадь прямоугольника

**Параметры:**

a (float) - длина прямоугольника
b (float) - ширина прямоугольника

**Возвращает:**
float - площадь прямоугольника

**Пример:**
area(5, 3)  
Возвращает 15

### triangle.py

Вычисляет площадь треугольника

#### area(a, b)

**Параметры:**

a (float): основание треугольника
h (float): высота треугольника
    
**Возвращает:**

float: площадь треугольника
    
**Примеры:**
area(5, 2.5)
6.25

#### perimeter(a, b, c):

Вычисляет периметр треугольника
    
**Параметры:**

a (float): первая сторона треугольника
b (float): вторая сторона треугольника
c (float): третья сторона треугольника
    
**Возвращает:**

float: периметр треугольника

**Примеры:**
perimeter(3, 4, 7)
14

### square.py

#### area(a)

Вычисляет площадь квадрата
    
**Параметры:**

a (float): сторона квадрата
    
**Возвращает:**

float: площадь квадрата
    
**Примеры:**
area(5.5)
30.25

        
#### perimeter(a)

Вычисляет периметр квадрата
    
**Параметры:**

a (float): сторона квадрата
    
**Возвращает:**

float: периметр квадрата
    
**Примеры:**
perimeter(3)
12.0
